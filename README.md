# Golt (Go + Bolt)
[![Godoc Reference](https://img.shields.io/badge/godoc-reference-blue.svg?style=flat)](https://godoc.org/github.com/tserkov/golt)
[![Build Status](https://img.shields.io/travis/tserkov/golt.svg?style=flat)](https://travis-ci.org/tserkov/golt)
[![Coverage Status](https://img.shields.io/coveralls/github/tserkov/golt.svg?style=flat)](https://coveralls.io/github/tserkov/golt?branch=master)
[![MIT License](https://img.shields.io/github/license/tserkov/golt.svg?style=flat)](../master/README.md)

A fully-featured [Bolt Protocol](https://boltprotocol.org/) driver for Go.

## Features
- [x] Basic client
- [ ] Basic server
- [x] Implementation of all types
- [x] Implementation of all messages
- [x] Message serialization
- [ ] Message deserialization
- [ ] Connection pooling
